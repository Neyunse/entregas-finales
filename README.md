## API Endpoints

### Products

* 'api/products/'(GET)
* 'api/products/:id'(GET)
* 'api/products'(POST)
* 'api/products/:id'(PUT)
* 'api/products/:id'(DELETE)

### Cart

* 'api/cart/:id'(DELETE)
* 'api/cart'(POST)
* 'api/cart'(GET)
* 'api/cart/:id/products/:id_prod' (POST) 
* 'api/cart/:id/products/:id_prod' (DELETE)
